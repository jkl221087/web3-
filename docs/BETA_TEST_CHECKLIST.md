# Beta Test Checklist

這份清單用來驗證目前服飾電商 DApp 在 Beta 階段的完整流程。

## 1. 測試前準備

- 確認前端可正常啟動
- 確認 Java backend 可正常啟動
- 確認合約地址、payment token 地址、chain id 已設定正確
- 確認測試錢包有足夠測試 USDT
- 準備至少 2 個角色地址
- `Admin / Seller`
- `Buyer`

---

## 2. 入口頁測試

- 開啟 `entry.html`
- 確認背景動畫、幣種裝飾、格線正常顯示
- 點 `Start` 後，資訊卡有正常展開
- 未連接錢包前，不應顯示個人資訊
- 連接錢包後，應顯示地址 / 角色 / session 狀態
- 連接成功後，`進入商店` 才出現
- `進入商店` 可正確進入首頁

---

## 3. 商品流程測試

### 商品建立

- 進入 `seller.html`
- 核准賣家身份後可正常看到商品表單
- 可填寫商品名稱、價格、分類、季節、款式、尺寸、顏色、庫存、描述
- 圖片可成功上傳
- 建立商品後，商品出現在商品管理列表
- 建立商品後，首頁商品區正常顯示
- 商品詳情頁正常顯示

### 商品編輯

- 點 `編輯` 後，會跳到編輯表單區塊
- 表單會正確帶入原始資料
- 修改後儲存成功
- 首頁 / 商品頁 / 商品管理頁同步更新

### 商品上下架

- 點 `下架` 後，商品管理頁顯示下架狀態
- 下架商品不應出現在首頁商品區
- 直接開 `product.html?id=...` 時應顯示已下架或不可購買
- 點 `重新上架` 後，首頁重新出現商品

### 商品刪除

- 點 `刪除` 後，出現站內確認框
- 取消刪除時商品仍存在
- 確認刪除後商品管理頁移除
- 首頁商品區同步移除
- 商品頁再訪問時不應顯示正常商品內容

---

## 4. 購物車流程測試

- 首頁商品卡可加入購物車
- 商品頁可選尺寸 / 顏色 / 數量後加入購物車
- 相同商品不同尺寸 / 顏色應分開成不同購物車項目
- 購物車可移除商品
- 可清空購物車
- `Bag` / `Orders` 按鈕位置正常
- 打開購物車時，`Bag / Orders` 會隱藏
- 關閉購物車後，按鈕重新出現
- 購物車總件數、總金額計算正確

---

## 5. 鏈上付款流程測試

- 未連接錢包時點 `前往結帳`，應先要求連接錢包
- 錢包已連接時，點 `前往結帳` 應走付款流程
- 若 USDT 餘額不足，應顯示 `transfer amount exceeds balance` 對應的友善錯誤
- 若 allowance 不足，應先觸發 `approve`
- `approve` 成功後，再建立訂單
- 訂單成功後：
- 購物車會清空
- 首頁訂單資料更新
- 後端 `orders` metadata 有寫入
- 鏈上 `order_info` 可查到資料

---

## 6. 買家流程測試

- 進入 `buyer-center.html`
- 可看到自己的訂單
- 搜尋 / 篩選功能正常
- 訂單明細顯示商品名稱、尺寸、顏色、價格正確
- 可執行收貨確認
- 收貨確認後，訂單狀態更新為可提領
- 收貨後可送出評價
- 收藏與最近瀏覽正常顯示

---

## 7. 賣家流程測試

- 進入 `seller-center.html`
- 只能看到自己接到的訂單
- 可更新物流狀態
- 已出貨
- 已到貨
- 提醒取貨
- 可看見可提領訂單
- 可執行提領
- 提領成功後：
- 賣家中心狀態更新
- payout history 寫入成功
- 提領紀錄可查詢

---

## 8. Admin 流程測試

- 進入 `admin.html`
- 非 admin 不可正常進入管理功能
- admin 可看到賣家申請
- admin 可核准賣家
- admin 可看商品治理摘要
- admin 可看異動紀錄 audit trail
- 商品管理與風險提醒資訊正常

---

## 9. 權限流程測試

- 未審核賣家不可建立商品
- seller 不能修改別人的商品
- seller 不能刪除別人的商品
- buyer 不能更新賣家物流狀態
- buyer 不能寫 seller payout history
- admin 可執行審核功能
- session 過期或未登入時，受保護 API 應正確拒絕

---

## 10. UI / 體驗測試

- 底部導覽列在所有主要頁面都置中
- `Bag / Orders` 在首頁位置正確
- 背景格線、動態裝飾、token glyph 顯示正常
- 頁面文字清晰可讀
- 商品卡不會過大或過小
- 響應式在手機尺寸下仍可用
- 空狀態卡片文案清楚
- 錯誤訊息對使用者可理解

---

## 11. 錯誤回歸測試

- `priceWei must be an unsigned integer string`
- `Product not found`
- `core.initWalletState is not a function`
- `paymentToken.balanceOf is not a function`
- `Method Not Allowed`
- `Not Found`
- `ERC20: transfer amount exceeds balance`

每次修正後，都要至少重跑一次對應流程。

---

## 12. Beta 出口條件

當以下條件成立，可視為 Beta 收斂完成：

- 商品 CRUD 全部正常
- 購物車與結帳正常
- 鏈上付款、收貨、提領正常
- 買家 / 賣家 / Admin 權限正常
- 主要頁面版型穩定
- 已知高頻錯誤都已回歸驗證
